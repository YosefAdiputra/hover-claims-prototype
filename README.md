# Hover Claims — AI-Powered Estimate Automation (Prototype)

A clickable prototype exploring how AI can draft Xactimate-ready repair scopes from Hover inspection data, with adjuster-in-the-loop review and full evidence grounding.

Built as a product challenge exercise for a Principal PM interview.

## What it demonstrates

- Claim queue dashboard with AI status and confidence scoring
- Three-column evidence review: line items, photo/3D/measurement evidence, confidence panel
- Plain-language explanations for every AI recommendation
- Edit flow with required reason codes for audit trail
- Exception handling for low-confidence items
- Final attestation and Xactimate submission

## Stack

- React 18 + Vite
- Tailwind CSS
- lucide-react icons
- Instrument Serif (display) + Geist (UI) from Google Fonts

## Run locally

    npm install
    npm run dev

Open http://localhost:5173
